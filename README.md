# CAPEX Investment Tracker

Sistema para controle e monitoramento de investimentos corporativos (CAPEX). Permite:
- Cadastro de projetos com orçamento aprovado
- Registro de movimentações (SAP, CAC, gastos)
- Cálculo de saldo, execução e desvio
- Dashboards de acompanhamento por área, projeto e período

## Estrutura do projeto
- **data/**: arquivos CSV de projetos e movimentações
- **notebooks/**: análises exploratórias e testes
- **src/**: código do dashboard e cálculos
- **docs/**: documentação e manuais

## Tecnologias
- Python
- Pandas
- Matplotlib / Plotly
- SQL
